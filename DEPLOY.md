# Deploy to Streamlit Community Cloud

1. Go to **[share.streamlit.io](https://share.streamlit.io)**
2. Sign in with GitHub
3. Click **"New app"**
4. Configure:
   - **Repository:** `meghana064/HR_Analytics`
   - **Branch:** `main`
   - **Main file path:** `employee_attrition_ai/app/streamlit_app.py`
   - **App URL:** (optional) e.g. `hr-analytics` → `https://hr-analytics.streamlit.app`
5. Click **"Deploy"**

Your app will be live in a few minutes.
